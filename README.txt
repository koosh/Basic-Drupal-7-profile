Basic Drupal installation profile
-----------------------------------------

This profile tries to set up a basic site to get a new project started with most
of the contributed modules (with the latest stable versions) already available
(see the basic.make file).

It is a Drupal standard profile copy, changes are:
 - no content types are made
 - no taxomies
 - user can register
 - Dashboard, Toolbar and Overlay modules are not enabled
 - Admin views, Admin menu toolbar and Module filter modules are enabled
 - Omega 3.1 theme is set as the default theme.
